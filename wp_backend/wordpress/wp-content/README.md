# Atlas Journal — wp-content

## Estrutura

- `mu-plugins/atlas-loader.php`: garante o carregamento do plugin principal
- `plugins/atlas-headless-core/`: plugin com campos, REST e endpoint da home
- `themes/atlas-headless/`: tema mínimo para manter o WordPress funcional

## O que já vem pronto

- campos meta para posts e páginas
- endpoint customizado para a home
- melhorias no REST API nativo
- CORS básico para frontend headless
- tema mínimo de fallback
