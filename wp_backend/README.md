# WordPress Backend

Pasta reservada para o backend em WordPress do projeto Atlas Journal.

## Papel do WP neste projeto

O WordPress será usado apenas como CMS/headless backend, responsável por:

- publicar posts
- gerenciar categorias e páginas
- cadastrar vídeos e destaques
- servir dados para o Next.js

## Estrutura sugerida

- instalação WordPress limpa
- tema mínimo só para backend, se necessário
- ACF opcional para campos como destaque, vídeo e blocos especiais
- WP REST API ou WPGraphQL

## Conteúdos iniciais

- posts
- categories
- page: Nossa História
- autores/colunistas
